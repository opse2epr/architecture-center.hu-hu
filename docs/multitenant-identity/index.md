## <a name="what-is-multitenancy"></a><span data-ttu-id="abff8-127">Mi az a több-bérlős mód?</span><span class="sxs-lookup"><span data-stu-id="abff8-127">What is multitenancy?</span></span>
<span data-ttu-id="abff8-128">A *bérlőt* felhasználók egy csoportja alkotja.</span><span class="sxs-lookup"><span data-stu-id="abff8-128">A *tenant* is a group of users.</span></span> <span data-ttu-id="abff8-129">Egy SaaS-alkalmazásban a bérlő az alkalmazás egyik előfizetőjét vagy ügyfelét jelenti.</span><span class="sxs-lookup"><span data-stu-id="abff8-129">In a SaaS application, the tenant is a subscriber or customer of the application.</span></span> <span data-ttu-id="abff8-130">A *több-bérlős* egy olyan architektúra, ahol egy alkalmazás egy fizikai példányán több bérlő osztozik.</span><span class="sxs-lookup"><span data-stu-id="abff8-130">*Multitenancy* is an architecture where multiple tenants share the same physical instance of the app.</span></span> <span data-ttu-id="abff8-131">Bár a bérlők osztoznak a fizikai erőforrásokon (például a virtuális gépeken vagy a tárterületen), minden bérlő saját logikai példánnyal rendelkezik az alkalmazásból.</span><span class="sxs-lookup"><span data-stu-id="abff8-131">Although tenants share physical resources (such as VMs or storage), each tenant gets its own logical instance of the app.</span></span>

<span data-ttu-id="abff8-134">Hasonlítsa össze ezt az architektúrát az egybérlős architektúrával, ahol minden egyes bérlő saját fizikai példánnyal rendelkezik.</span><span class="sxs-lookup"><span data-stu-id="abff8-134">Compare this architecture with a single-tenant architecture, where each tenant has a dedicated physical instance.</span></span> <span data-ttu-id="abff8-135">Egybérlős architektúra esetén bérlők hozzáadásakor új példányokat kell indítani az alkalmazásból.</span><span class="sxs-lookup"><span data-stu-id="abff8-135">In a single-tenant architecture, you add tenants by spinning up new instances of the app.</span></span>

## <a name="identity-in-a-multitenant-app"></a><span data-ttu-id="abff8-148">Identitás több-bérlős alkalmazásokban</span><span class="sxs-lookup"><span data-stu-id="abff8-148">Identity in a multitenant app</span></span>
