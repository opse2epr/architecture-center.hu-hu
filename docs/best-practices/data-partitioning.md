## <a name="offline-migration"></a><span data-ttu-id="221cb-834">Offline áttelepítés</span><span class="sxs-lookup"><span data-stu-id="221cb-834">Offline migration</span></span>
<span data-ttu-id="221cb-835">Offline áttelepítés késései a legegyszerűbb módszere mivel csökkenti a veszélyét annak, hogy a versengés lépett fel.</span><span class="sxs-lookup"><span data-stu-id="221cb-835">Offline migration is arguably the simplest approach because it reduces the chances of contention occurring.</span></span> <span data-ttu-id="221cb-836">Nem módosítja az adatokat, amíg folyamatban van áthelyezése és szerkezetátalakítás.</span><span class="sxs-lookup"><span data-stu-id="221cb-836">Don't make any changes to the data while it is being moved and restructured.</span></span>

<span data-ttu-id="221cb-837">Ez a folyamat elméleti szinten, a következő lépéseket tartalmazza:</span><span class="sxs-lookup"><span data-stu-id="221cb-837">Conceptually, this process includes the following steps:</span></span>

## <a name="online-migration"></a><span data-ttu-id="221cb-845">Online áttelepítése</span><span class="sxs-lookup"><span data-stu-id="221cb-845">Online migration</span></span>
