# <a name="aws-to-azure-services-comparison"></a>Azure-szolgáltatások összehasonlítása AWS

Ez a cikk segít megérteni, hogyan hasonlítsa össze a Microsoft Azure-szolgáltatások az Amazon Web Services (AWS). Egy Azure és az AWS multicloud megoldás tervezésének részeként, és az Azure-bA áttelepítése, összehasonlíthatja kategóriából Azure és a AWS szolgáltatások informatikai képességeit.

Az alábbi táblázatban vannak bizonyos AWS szolgáltatások felsorolt több Azure-szolgáltatásokhoz. Az Azure-szolgáltatások egy másik hasonló, de és képességek mélysége eltérők lehetnek.

[Töltse le az Azure & AWS felhő Szolgáltatástérkép PDF](https://aka.ms/awsazureguide)

## <a name="azure-and-aws-for-multicloud-solutions"></a>Az Azure és az AWS multicloud megoldások

A kezdő nyilvános felhő platform, mint az Azure és az AWS minden egyes vállalatok számára egy széleskörű és részletes képességeket kínál és globális szoftvertermékekre. Számos szervezet még jelölje ki, a nagyobb rugalmasságának és, valamint terjednek a kockázat és a függőségek egy multicloud módszert használja, a két platform. A vállalatok és a szoftverforgalmazók tanácsadás például is létrehozásához, és az Azure és a AWS, ezekről a platformokról határoz meg a felhő piaci igény szerint a legtöbb.

Az áttekintést az Azure AWS felhasználók számára, lásd: [Bevezetés az Azure-bA az AWS szakértők](index.md).
